# Конспект по первому модулю + записи из "Введение"
### CLI GIT

```bash
git status  #показывает статус репо

git add . #делает доступными для commit изменения

gyt sync #покажет изменения

git commit -m "comment" #делает commit

git push #пушит коммит в репо

git pull #синхронизируется с удаленным репо

git checkout -b "type/name" #создает ветку

git push --set-upstream origin type/name # подключает ветки к удаленному репозиторию
```
